# Examples

This directory contains trimmed Gaussian fixtures and generated demo output.
The fixture logs are intentionally small and keep only the Gaussian sections parsed by Quantum Analyzer.

## Fixtures

| File | Molecule | Basis | Purpose |
| --- | --- | --- | --- |
| `fixtures/methane_6-31g.log` | methane | 6-31G | Main parser, matrix and quantum-analysis fixture. |
| `fixtures/methane_sto-3g.log` | methane | STO-3G | Second methane calculation for grouped/pivot reports. |
| `fixtures/water_sto-3g.log` | water | STO-3G | Different molecule fixture for grouping tests. |

## Direct Analyzer Demo

| File | Purpose |
| --- | --- |
| `analysis_demo.lua` | Analysis template with grouped pivot tables. |
| `generated/analysis_reports/*.md` | Generated grouped Markdown reports, one file per molecule group. |

Regenerate the direct Markdown report:

```bash
xmake run examples
```

Useful built-in console templates:

| Template | Purpose |
| --- | --- |
| `standard_analysis` | Detailed report for one or more Gaussian files. |
| `summary_analysis` | Grouped basis-comparison report. |
| `single_molecule_overview` | Compact one-molecule summary. |
| `method_comparison_analysis` | Mulliken/Lowdin comparison across basis sets. |
| `lone_pair_analysis` | Lone-pair oriented analysis. |

## Pandoc Code Block Demo

| File | Purpose |
| --- | --- |
| `pandoc_demo.md` | Pandoc source document with a `.qa` code block. |
| `generated/pandoc_demo.html` | HTML generated by Pandoc after executing the code block. |
| `generated/pandoc_overlap_atoms_heatmap.svg` | Atom-grouped overlap heatmap generated during Pandoc rendering. |
| `generated/pandoc_mulliken_charges.svg` | Mulliken charge bar chart generated during Pandoc rendering. |

Regenerate the Pandoc demo:

```bash
xmake run examples
```

Run the usual smoke checks:

```bash
xmake run test-all
```
